# 📝 To-Do List App

A simple and responsive To-Do List web app built using HTML, CSS, and JavaScript.

---

## 🚀 Features
- ➕ Add tasks
- ✔ Mark tasks as completed
- 🗑 Delete tasks
- 🔍 Filter tasks (All / Completed / Pending)
- 📊 Progress bar tracking
- 💾 Data saved using Local Storage
- 🌙 Dark mode support

## 🛠 Technologies Used
