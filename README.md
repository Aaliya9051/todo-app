# 📝 To-Do List Web App

A modern and responsive To-Do List application built using HTML, CSS, and JavaScript.  
This app helps users manage daily tasks efficiently with progress tracking and a clean user interface.

## ✨ Features

- ➕ Add new tasks
- ✔ Mark tasks as completed (icon-based toggle)
- 🗑 Delete tasks
- 🔍 Filter tasks:
  - All
  - Completed
  - Pending
- 📊 Dynamic progress bar (auto updates)
- 💾 Local Storage support (data persists after refresh)
- 🌙 Dark mode toggle
- 🤖 Smart task suggestions
- 🎨 Clean and responsive UI

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

## ⚙️ How It Works

- Tasks are stored in browser **Local Storage**
- Clicking the icon toggles task completion
- Progress bar updates automatically based on completed tasks
- Filters help organize tasks efficiently
